# gite

A Node CLI with some helpful commands related to git.

Built using [oclif](https://oclif.io/).

## Usage

```bash
$ gite
$ gite open
$ gite help
$ gite help open
$ gite open -h
$ gite open --help
```

## Commands
### open
```bash
$ open [PATH]
```
Opens the Opens GitHub for the repository associated to a directory in the default web browser.

## Running locally
```bash
$ ./bin/run
$ ./bin/run open
```
